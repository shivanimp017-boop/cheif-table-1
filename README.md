# 🍽️ Chef's Table — AI Recipe Assistant

An AI-powered recipe app with food detection, personalized recommendations and reinforcement learning based suggestions.

## Features
- 🔐 User Authentication (Register/Login)
- 🔍 Smart Recipe Search (TheMealDB API)
- 📷 Food Detection via Camera/Upload
- 🥦 Veg / 🍖 Non-Veg / ⭐ Favourites
- 👤 User Profile with Search History
- ➕ Add Your Own Recipes
- 🤖 AI-powered Recommendations (RL)

## Tech Stack
- Python 3.14 + Flask
- TheMealDB API
- HTML/CSS/JavaScript
- Reinforcement Learning (Q-Learning)

## Setup
```bash
pip install -r requirements.txt
python app.py
```
Open `http://localhost:5000`
